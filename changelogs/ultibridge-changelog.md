# Changelog

All notable changes to **UltiBridge** will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-03-15
### Added
- **Initial Release** of UltiBridge.
- **Unreal Engine Bridge**:
    - Direct export to Unreal Content Browser via Python remote execution.
    - Automated LOD generation and export.
    - Automatic material instance matching.
    - Support for Static Meshes, Skeletal Meshes, and Animations.
    - Collision mesh generation (UCX/UBX).
- **Unity Bridge** (Beta):
    - FBX export presets for Unity coordinate system.
    - Prefab generation helper.
- **Batch Processing**:
    - Multi-object export capabilities.
    - Naming convention enforcement tools.
- **UI**:
    - Unified panel for all bridge settings.
    - Project path configuration management.
